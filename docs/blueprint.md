# **App Name**: DocXtoExcel

## Core Features:

- DocX Upload: Allow users to upload a .docx file from their local machine.
- Content Extraction: Extract question text, options (A, B, C, D), and images from the uploaded .docx file.
- Excel Generation: Generate an Excel file with columns for Sr. No, Question content, Alternative1, Alternative2, Alternative3, Alternative4.
- Text Wrapping: Enable text wrapping for all cells in the generated Excel file.
- Column Width Adjustment: Set fixed column widths for Question content (wide) and Alternatives (equal width).
- Borders: Add borders to all cells in the Excel file.
- Image Embedding: Embed extracted images at correct positions within the generated Excel file.

## Style Guidelines:

- Primary color: Soft blue (#A0CFEC) to give a professional and calm feel.
- Background color: Light grey (#F0F0F0) to ensure content is easily readable.
- Accent color: Light Green (#B5E487) for CTAs and highlights to represent smooth conversion.
- Body and headline font: 'Inter', sans-serif, for a modern, machined, objective, neutral look that can support legibility for long lines of text.
- Use clear and concise labels for input fields and buttons.
- Maintain a consistent margin and padding around elements.
